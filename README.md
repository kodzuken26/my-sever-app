Доброе время суток! Для того чтобы запустить мой проект нужно использовать команду npm run dev. После запуска перейдите по ссылке http://localhost:3000. 
После этого перед вами окажется главная страница сайта. На ней представлен каталог товаров и строка поиска. При вводе в поиске слова нужно нажать на клавишу Enter и после этого выйдет результаты поиска, содержащие то, что вы ввели. 
При нажатии на товар, вы перейдете на страницу товара, где будет представлена подробная информация о нем. При нажатии на кнопку "Поделиться" у пользователя будет возможность скопировать ссылку товара или отправить ее в приложение. Ниже информации о товаре представлен слайдер, кнопки у него появляются при уменьшении ширины экрана, как было показано в макете. 
Весь сайт адаптируется под указанные в макете размеры. 
This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
